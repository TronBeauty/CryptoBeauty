## Development

### Install Dependencies

```
npm install
```

### Test contract

```
truffle test
```

### Frontend Test

More instructions are in the README under `client` directory.

```
cd client
npm install
npm start
```

### To build and publish
(`npm install gh-pages -g` if haven't installed github pages)
```
cd client
npm run build
gh-pages -d build 
```
